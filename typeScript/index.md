# What is TypeScript?
TypeScript is a superset of JavaScript created by Microsoft that adds static typing and other powerful features.

👉 In simple words:

TypeScript = JavaScript + Types + Better Developer Experience

It compiles down to normal JavaScript, so browsers understand it.

🧠 Why TypeScript?
Catch errors before runtime
Better auto-completion & IntelliSense
Easier to maintain large applications
Improves code readability
