# How do you monitor production?

We monitor production in five layers:

- User behavior analytics (Google Analytics / Mixpanel)
- Web performance monitoring (Lighthouse / Core Web Vitals)
- Frontend error monitoring (Sentry / LogRocket)
- Backend & API monitoring (CloudWatch / Datadog)
- Full system observability (Datadog / New Relic)

This gives us complete visibility from user interaction to backend failures.
