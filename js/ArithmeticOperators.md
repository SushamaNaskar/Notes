# Unary Operator
- Operators that use one values are called Unary operators.
- !

# Binary Operator
- Operators that use two values are called binary operators.
- <(greater than),>(less than),  >= (greater than or equal to), <= (less than or equal to), == (equal to), and != (not equal to).

# Ternary Operator/ conditional operator
- Operators that use three values are called Ternary operators.
- true ? 1 : 2
