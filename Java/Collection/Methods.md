# 🔹 Common Methods (Collection Interface)

| Method     | Use            |
| ---------- | -------------- |
| add()      | Add element    |
| remove()   | Remove element |
| size()     | Count          |
| contains() | Check          |
| isEmpty()  | Check empty    |
| clear()    | Remove all     |


# Time Complexity Cheat Sheet

| Structure  | Search   | Insert   | Delete   |
| ---------- | -------- | -------- | -------- |
| ArrayList  | O(1)     | O(n)     | O(n)     |
| LinkedList | O(n)     | O(1)     | O(1)     |
| HashSet    | O(1)     | O(1)     | O(1)     |
| TreeSet    | O(log n) | O(log n) | O(log n) |
| HashMap    | O(1)     | O(1)     | O(1)     |
