# Key features of Java 8 include:
1. Lambda Expressions
2. Stream API
3. Functional Interfaces
4. Method References
5. Default and Static Methods in Interfaces
6. Optional Class
7. New Date and Time API
8. Concurrency Enhancements
9. JVM Improvements (Metaspace)
10. Nashorn JavaScript Engine : Run JavaScript inside Java (now deprecated in later versions)



# ✅ Best Learning Order
- Functional Interfaces → define rule (foundation) 
- Lambda Expressions → implement (how you implement them)
- Method References → shorten (shortcut of lambda)
- Stream API → use on data (where you use lambdas heavily)
- Default & Static Methods → enhance interface (supporting feature in interfaces)


👉 “First define the rule → write logic → shorten it → use it on data → enhance interface”
