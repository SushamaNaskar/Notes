# Before Cloud:
Before cloud computing, companies had to buy and maintain their own physical servers in their offices or data centers. This required large upfront costs, space for hardware, electricity, cooling systems, and dedicated IT teams. If traffic increased, they had to purchase more servers, which took time and money. Many servers also stayed underused during low traffic periods.


# cloud
Clouds are basically servers (machines) owned and maintained by cloud providers like companies that run large data centers. Instead of buying and managing your own hardware, you can rent these servers over the internet and pay only for the resources or services you use. This is called the pay-as-you-go model.

Cloud services (service models) are usually grouped into three main types:

SaaS (Software as a Service) – Provides ready-to-use software over the internet (like email apps, document tools, etc.).

IaaS (Infrastructure as a Service) – Provides basic infrastructure like virtual machines, storage, and networking.

PaaS (Platform as a Service) – Provides a platform to build, run, and deploy applications without managing servers.



# cloud computing
- Cloud Computing is the ability to store data, process it, and access applications using remote servers over the internet from anywhere in the world instead of using your own computer or local server.
